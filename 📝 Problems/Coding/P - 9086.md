## 🧠 접근

### 첫 생각
- 이건 t번 반복할 정수를 먼저 받아야겠다.
- 그리고 string안에 들어갈 문자열 하나 만들고.
- 그리고 반복문으로 때려서 t번 string을 받으면 되겠지?
- 근데 맨 앞과 뒤를 인덱싱해서 빼와야하나? 그럴려면 몇자린지 알아야하는데?
- 그럼 이건 뒤로 미루고. 그 다음은 맨 앞과 맨 뒤만 알 수 있는 함수가 있을지도?

### 핵심 아이디어
- 그래서 나온 함수가? stringname.front(), stringname.back()이다~
- 이렇게 해서 맨 앞, 맨 뒤만 출력하면 되지롱~

## ✏️ 풀이

### C++
```cpp
#include <iostream>
#include <string>

auto main() -> int {
  std::cin.tie(nullptr)->sync_with_stdio(false);

  int t;
  std::string cases;
  if (std::cin >> t) {
    while (t--) {
      if (std::cin >> cases) {
        std::cout << cases.front() << cases.back() << "\n";
      }
    }
  }
}

```

- 시간: O()
- 공간: O()
