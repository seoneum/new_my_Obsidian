## 🧠 접근

### 첫 생각
- 일단 int로 개수 받자. 그리고 string크기 설정해서 s로 받아.
- 이걸 어떻게 할까? 아무래도 ASCII변환 해야하겠지?
- 근데 어떻게? 숫자를 ASCII변환하는데 그걸 char=int로 어떻게 하지? 
- 그냥 (int)s하면 안될 것 같은데

### 핵심 아이디어
- string s의 모든 배열 원소들을 for문으로 돌려서 하나마다 `- '0'`를 해주면 된다!
- **std::accumulate** 사용하기!
  ```cpp
  #include <numeric>
  std::accumulate(시작, 끝, 시작 값, 계산 방법);
  //ramda 함수 사용
  [캡쳐](입력 변수){실행 코드}
  [캡쳐] : 외부 변수를 가져옴
  (입력 변수) : accumulate가 람다에게 값 2개 던짐.
  {계산 로직} : 반드시 다음 단계의 누적 값을 리턴해야함.
  ```

## ✏️ 풀이

### C++
```cpp
#include <iostream>
#include <numeric>
#include <string>

auto main() -> int {
  int n, sum = 0;
  std::string m;
  if (std::cin >> n) {
    if (std::cin >> m) {
      for (char c : m) {
        sum += (c - '0');
      }
      std::cout << sum;
    }
  }
}
// 첫 실패한 이유 : sum을 초기화 하지 않았음.
//  뭔가 이상하다.
//  하지만 정석임.
// modern
auto main() -> int {
  std::cin.tie(nullptr)->sync_with_stdio(false);

  int n;
  std::string s;
  if (std::cin >> n) {
    if (std::cin >> s) {
      int sum = std::accumulate(s.begin(), s.end(), 0, [](int acc, char c) {
        return acc + (c - '0');
      }); // string s의 시작점 = s.begin(), 끝 : s.end(), acc = 더한 누적.
      std::cout << sum;
    }
  }
}

```

### 복잡도
- 시간: O()
- 공간: O()

```cpp
// 원리: (1+'0') + (2+'0') + ... = (1+2+...) + (N * '0')
int sum = std::accumulate(s.begin(), s.end(), 0) - (s.size() * '0');
```
