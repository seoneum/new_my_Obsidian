# B - 서양 철학사(초독)

## Notes
- 모르는 단어
	1. 서론
		- [[Joseph Stailn]] : 구소련 독재자. 공산주의는 당을 숭배하는 국가 숭배 주의로 봐야함. -> 공산주의의 꼭대기에 있는 스탈린이 변증법적 유물론을 마음대로 해석하더라도 스탈린의 말이 새로운 진리라고 생각. -> 미친 독재로 이어짐
		- 변증법적 유물론 : 공산주의의 절대 진리.
		- 사변적 : 경험이나 현실에 기반하지 않고 순수 이성이나 관념만으로 깊이 생각하거나 추론하는 태도나 이론
		- 아리우스 주의 : 4세기 초 아리우스가 주장한 반 삼위일체적 신학
		- 튜튼족 : 고대 중부 유럽에 존재했던 게르만의 한 계통. 로마를 개패듯이 팸. 
		- 프란치스코회 탁발수도자 : 성프란치스코가 창립한 가톡릭 수도회 소속 수도자.
		  재산 포기, 맨발로 다니며 구걸하면서 살아감. 중세에 교황이랑 사이 별로.
		- [[Immanuel Kant]] : 각자 세상을 자신의 마음으로 구성해서 본다. 하지만 물자체는 존재함
		- [[Johann Gottlieb Fichte]] : 물자체는 없어. 내가 세상의 창조자다. 자아가 세상을 만든다.
		- 철학적 주관주의 : 루소, 피히테같이 객관적 진리 없음. 내가 느끼고 생각하고 바라는 것이 진리다.
		- 정치적 무정부주의 : 철학적 주관주의에 의해 내 생각과 의지가 최고의 법이라면 국가/법률은 장애물이다. (주관주의 + 무정부주의 -> 독재[^1])
		- 국가 숭배 주의 : 주관주의/무정부주의의 반작용으로 나옴. 개인이 너무 깝치네. 국가가 신이다. (역사적 흐름 : 중세=교황 힘⬇️ -> 주관주의 대두 -> 혼란 -> 국가숭배). 대표적으로 [[Georg Wilhelm Friedrich Hegel|Hegel]]이 있음. 이놈은 "국가는 지상 위를 걸어가는 신이다" 라고 함. = 개인은 국가를 위해 존재할 때만 가치가 있다. -> 전체주의와 전쟁으로 귀결됐다고 생각함. 
		- [[Voltaire]] : 계몽주의 철학자. 무지하니 맹신하지 말고 존교적 관용 배풀고 이성적으로 사회를 고쳐나가자!! [[Johann Gottlieb Fichte]], [[Jean-Jacques Rousseau]]와 정반대에 위치해있음. 하지만 신에게 불경한 태도였나? 이런 뉘앙스의 구절이 있네.
		- [[Homer]]의 소설 속 신들은 인간과 다르지 않음. 도덕적으로도 인간과 비슷하다. 세계를 창조한 것도 아님. 몇몇만 하고 나머지는 정복만 일삼고 창조적인 이야기 안 함.
		- 아테 : 그리스 신화에 나오는 불행과 재앙의 여신. 불행/재앙을 뜻하기도 함.
		- 차라투스트라 : 조로아스터임. [[Friedrich Nietzsche]]가 말한 차라투스트라가 여기서 따온 것임. 실제 역사에서 처음으로 선과 악을 구분한 사람으로 알려짐. 
		- [[Xenophanes]] : 호메로스 신화 존나 깜. 논리적으로 이게 말이 되냐? 이성적으로 말이 안되는 이상한 신앙을 깠다. "소가 신을 그린다면?"[^2]의 비유가 있음. 러셀은 이 사람을 "미신에서 이성으로 넘어가는 다리"로 생각함.
		- 문명인과 야만인을 구별하는 기준은 주로 사려[^3]이다.
	2. 고대 철학 - 소크라테스 이전
		
- 궁금증 + genimi 답변
	1. 
		- 개신교와 무정부주의가 맥락을 같이 한다는 말을 이해하지 못하겠었음
			-> 가톨릭 : 진리는 교회에. <-> 개신교 : 진리는 자신의 마음에.
			-> 진리가 나의 양심에 있다고? 그럼 내 양심에 따라 행동하면 누구도 나에게 명령할 수 없잖아. -> 정부야 간섭하지마 = 무정부주의.
		- 감수성을 예찬하는 시대의 낭만주의 철학을 왜 부정적으로 표현하지?
			- 러셀은 이성>>>>감성임. 
			1) 기준이 사라져 대화 안됨 : 이성은 객관적이지만 감성은 너무 주관적임.
			2) '동정심'은 쉽게 '잔혹함'으로 변질됨 : 감정의 호소하는 도덕? 1차원적으론 좋다 이거야. 근데 "우리 민족 불쌍해" -> "우리 민족을 괴롭히는 저놈들 다 죽여" 광기가 됨.
	2. 고대 철학 - 소크라테스 이전
