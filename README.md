# 📚 CMDS - 대학생을 위한 Obsidian Vault 시스템

> **Connect → Merge → Develop → Share**
> 
> 흩어진 지식을 연결하고, 통합하고, 발전시키고, 공유하는 학습 시스템

## 🎯 이 가이드의 목적

이 가이드는 대학생이 Obsidian을 활용해 **체계적인 지식 관리 시스템**을 구축할 수 있도록 돕습니다.

### 이런 분들에게 추천합니다:
- 📝 강의 노트, 과제, 시험 준비를 한 곳에서 관리하고 싶은 분
- 🔗 배운 내용들 사이의 연결고리를 만들고 싶은 분
- 🃏 플래시카드로 효율적인 암기 학습이 필요한 분
- 📅 일일/주간 회고로 학습을 점검하고 싶은 분

## 🏷️ Prefix 체계

파일명 앞에 Prefix를 붙여 파일 종류를 즉시 파악할 수 있습니다.

| Prefix | 용도 | 예시 |
|--------|------|------|
| `D -` | Daily Note (일일 기록) | `D - 2024-03-15` |
| `N -` | Note (일반 메모) | `N - 아이디어 메모` |
| `L -` | Lecture (강의 노트) | `L - 자료구조 3주차` |
| `C -` | Concept (개념 정리) | `C - 빅오 표기법` |
| `P -` | Problem (문제 풀이) | `P - 백준 1234` |
| `MTG -` | Meeting (회의록) | `MTG - 2024-03-15 팀미팅` |
| `Q -` | Question (미해결 질문) | `Q - 왜 시간복잡도가?` |
| `FC -` | FlashCard (암기 카드) | `FC - 자료구조 기초` |
| `FYN -` | Feynman (파인만 노트) | `FYN - 2024-03-15` |
| `JRN -` | Journal (저널/회고) | `JRN - 2024-03-15` |
| `PRJ -` | Project (프로젝트) | `PRJ - 캡스톤 디자인` |
| `R -` | Reference (참고자료) | `R - Git 명령어 정리` |

## 🃏 FlashCard 작성법

Spaced Repetition 플러그인과 호환되는 형식입니다.

### 기본 형식
```markdown
질문 내용
?
답변 내용
```

### 예시
```markdown
빅오 표기법에서 O(n)의 의미는?
?
입력 크기 n에 비례하여 선형적으로 시간이 증가하는 알고리즘

---

스택(Stack)의 특징은?
?
LIFO (Last In First Out) - 후입선출 구조
```

### 분야별 FC 파일 구성 권장
- `FC - [과목명] 기초.md` - 기본 개념
- `FC - [과목명] 중급.md` - 심화 내용
- `FC - [과목명] 시험대비.md` - 시험 직전 정리

## 📝 워크플로우

### 일일 루틴
1. **아침**: Daily Note 생성 → 오늘 할 일 정리
2. **수업 중**: `L -` 강의 노트 작성
3. **수업 후**: 중요 개념 → `C -` 개념 노트로 정리
4. **복습**: 암기할 내용 → `FC -` 플래시카드 추가
5. **저녁**: Daily Note에 오늘 배운 것 정리

### 주간 루틴
1. Weekly Review에서 한 주 회고
2. 미완료 할 일 점검
3. 다음 주 계획 수립
4. 플래시카드 복습 세션

### 학기 단위
1. 학기 시작: 과목별 폴더 생성
2. 중간고사 전: 개념 노트 → 플래시카드 변환
3. 기말고사 전: Feynman 기법으로 총정리
4. 학기 종료: 아카이브 정리

## 🧠 학습 기법 통합

### 1. 파인만 기법 (Feynman Technique)
- `FYN -` 노트에서 개념을 **쉬운 말로** 설명
- 막히는 부분 = 이해 부족 → 다시 공부

### 2. 능동적 회상 (Active Recall)
- 플래시카드로 자신을 테스트
- 정답 보기 전에 먼저 떠올려보기

### 3. 간격 반복 (Spaced Repetition)
- Spaced Repetition 플러그인 활용
- 잊을 만할 때 다시 복습

### 4. 연결 학습
- `[[링크]]`로 관련 개념 연결
- Graph View로 지식 지도 시각화
