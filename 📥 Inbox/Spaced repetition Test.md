## 📂 옵시디언 플러그인 마스터 카드

### 1. UI 및 편집 도구 (1~4)

- Outliner 적용 후 `ctrl + A`를 누르면 선택되는 범위의 단계는? :: 문장 -> 문단 -> 전체 위계 순서
    
- 사용한 용어를 기반으로 자동완성 기능을 제공하는 플러그인은? :: ==various Complements==
    
- Editing toolbar에서 모든 마크다운 양식을 지워주는 아이콘 모양은? :: 지우개 모양
    
- Highlightr에서 드래그 후 색상을 지정하는 단축키 조합은?
    
    ?
    
    ctrl + p 누른 후 색상 지정
    

### 2. 관리 및 자동화 (6~9)

- Tag wrangler에서 태그 이름을 한꺼번에 변경하는 방법은? :: 오른쪽 바의 태그 목록에서 `rename` 사용
    
- `refactor:split note by headings` 명령어의 역할은? :: 헤딩(Heading)마다 노트를 분리시킴
    
- Templater: 새 노트를 템플릿으로부터 생성하는 단축키는? :: `Alt + n`
    
- Templater: 활성 파일에서 템플릿을 교체(Replace)하는 단축키는? :: `ctrl + alt + t`
    
- 캘린더에서 데일리 노트를 생성할 때 템플릿이 자동 적용되게 하려면 어디서 설정하나? :: `Periodic Note` 설정
    

### 3. 정보 수집 및 북마크 (10~13)

- URL 복사 후 제목으로 쓸 문장을 드래그한 상태에서 바로 링크를 입히는 키는? :: `ctrl + v`
    
- 국내 도서 정보를 정리할 때 사용하는 명령어 키워드는? :: `ctrl + p` 실행 후 `korea` 검색
    
- Book Search 기능을 실행하는 단축키는? :: `ctrl + shift + b`
    
- Obsidian Web Clipper에서 유료 API 비용을 아끼기 위해 무료일 때만 사용을 권장하는 인터프리터는? :: ==Gemini==
    

### 4. 데이터 및 시각화 (14~16)

- 속성(Properties)이나 태그를 데이터베이스화하여 정렬 및 조회를 가능하게 하는 플러그인은? :: ==Data view==
    
- Canvas에서 여러 요소(노트, 이미지 등)를 묶어 함께 움직이게 만드는 방법은? :: 그룹(Group) 만들기
    
- Excalidraw에서 사이즈나 방향 조절 시 각도를 고정시키는 보조 키는? :: `Shift`
    

### 5. 협업 및 검색 (18~20)

- 칸반(Kanban) 보드를 새로 만드는 방법은? :: `ctrl + p` -> `kanban` 입력하여 새 보드 생성
    
- Share Note 플러그인으로 노트를 웹에 공유하기 전 설정해야 하는 것은? :: Option에서 `connect plug`를 외부 프로그램으로 열기
    
- 구글 검색 결과 옆에 내 옵시디언 노트를 함께 띄워주는 플러그인은? :: ==omniserch== (HTTP Server ON 필요)
    

### 6. 분석 및 스타일 (21~26)

- 그래프 뷰에서 현재 파일과 연결된 관계만 따로 보는 기능은? :: `local graph` (`ctrl + p`로 실행)
    
- Graph Analysis에서 노드 간 가중치 계산을 위해 켜두어야 할 알고리즘 2가지는? :: `Adamic`, `jaccard`
    
- 특정 노트가 다른 곳에서 몇 번이나 참조되었는지 숫자로 보여주는 플러그인은? :: ==Strange New World==
    
- Minimal Theme의 세부 스타일을 커스텀할 때 필요한 플러그인은? :: `Style Setting` 및 `Minimal Theme Setting`
    
- 태그나 경로 등 정해진 기준에 따라 링크 색상을 바꿔주는 플러그인은? :: ==Supercharged Links==
    

### 7. 생산성 관리 (27~29)

- 노션(Notion)처럼 라이트하고 직관적인 환경을 만들어주는 플러그인 대안은? :: `Make.md`
    
- Workspace Plus에서 현재 작업 중인 디스플레이 배치를 저장하는 단축키는? :: `ctrl + u`
    
- 현재 공부 중인 이 암기 시스템을 구현하는 플러그인 이름은? :: ==Spaced Repetition==
