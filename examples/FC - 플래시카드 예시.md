# FC - 플래시카드 예시

> 📌 이 파일은 플래시카드 작성 예시입니다.
> 
> **Spaced Repetition 플러그인 설정**:
> - 플래시카드 구분자: `?`
> - 카드 사이 구분: `---`

---

## 🃏 예시 카드들

빅오 표기법에서 O(1)의 의미는?
?
입력 크기와 관계없이 항상 일정한 시간이 걸리는 알고리즘
예: 배열에서 인덱스로 접근

---

스택(Stack)의 특징은?
?
LIFO (Last In First Out)
- 후입선출 구조
- push: 삽입, pop: 제거
- 예: 브라우저 뒤로가기, 실행취소(Undo)

---

HTTP 상태 코드 200의 의미는?
?
OK - 요청이 성공적으로 처리됨

---

Git에서 `git pull`은 무엇을 하는 명령어인가?
?
원격 저장소의 변경사항을 로컬로 가져와 병합
= `git fetch` + `git merge`

---

재귀(Recursion)의 필수 요소 2가지는?
?
1. **Base Case**: 종료 조건
2. **Recursive Case**: 자기 자신을 호출하는 부분

---

## 📝 작성 팁

1. **질문은 구체적으로**
   - ❌ "포인터란?"
   - ✅ "C언어에서 포인터의 정의와 용도는?"

2. **답변은 핵심만**
   - 너무 길면 암기하기 어려움
   - 키워드 중심으로 작성

3. **예시 포함**
   - 추상적 개념은 예시와 함께

4. **카드 분리**
   - 한 카드에 여러 개념 X
   - 하나의 질문 = 하나의 핵심 답변
