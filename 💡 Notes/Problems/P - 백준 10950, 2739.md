- Ranges Header
	- 파이프 문법 지원
		- 데이터 처리를 이어붙여서 함.
	- 컨테이너 자체를 하나의 범위로 인식. 
```cpp
//파이프 문법 예문
for (int i : data){
	if (i%2==0){
		result.push_back(i*i);
	}
}
->
auto result = data
	| std::views::filter([](int n){return n%2 == 0;})
	| std::views::transform([](int n){return n*n;});
//중간 데이터를 저장하는 임시 메모리 생성되지 않음. 순서대로 직관적임.


//범위 인식 예문
std::vector<int> v = {1,2,3,4,5};
std::sort(v.begin(), v.end());
->
std::vector<int> v = {1,2,3,4,5};
std::ranges::sort(v);
```
- std::views
	- 데이터를 소유하지 않고 **바라보는 방식**만 정의
	- **지연 평가(Lazy Evaluation)** : 데이터 미리 계산 X. 호출 시 그 때 그 때 주기.
- #25314 영수증 문제
	- 이 문제도 무난하게 풀었다.
	- 그냥 간단히 if와 for을 쓰면 되는 문제.
```cpp
#include <iostream>
#include <ranges>

auto main() -> int {
  std::cin.tie(nullptr)->sync_with_stdio(false);

  int t_cost, k_num, a, b;
  int sum = 0; //int a, b =0이면 a는 아무 값X, b = 0이라는 뜻.
  if (std::cin >> t_cost >> k_num) {
    for ([[maybe_unused]] int i : std::views::iota(1, k_num + 1)) {
    //[[maybe_unused]]는 안 쓰는데 명시적으로는 필요해서 기입한 것.
      if (std::cin >> a >> b) {
        sum += a * b;
      }
    }
  }
  if (sum == t_cost) {
    std::cout << "Yes";
  } else {
    std::cout << "No";
  }
  return 0;
}
```
- 25314번
```cpp
#include <iostream>
#include <ranges>

auto main() -> int {
  int n;
  if (std::cin >> n) {
    for (auto str : std::views::repeat("long ") | std::views::take(n / 4)) {
      std::cout << str;
    }
    std::cout << "int";
  }
  return 0;
}

auto str = 그냥 넣는 것
std::views::repeat("long ") : repeat()안의 문자열을 무한 생성
std::views::take(n/4) : 무한개 중에서 앞의 n/4개만 가져옴. 
```
