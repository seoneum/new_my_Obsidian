# 효과적인 독서법과 기억력 향상 전략

> **한 줄 요약**: 읽는 것보다 **출력(글쓰기, 설명)**이 중요하고, **핵심 키워드만** 메모하며, **구조화**와 **반복**으로 장기 기억화한다.

---

## 📖 왜 책을 읽어도 기억이 안 날까?

> [!important] 뇌의 특성
> - 하루 만에 읽은 내용의 **70~90%**가 뇌에서 사라진다
> - 이건 우리 잘못이 아니라 뇌가 사용하지 않는 정보를 정리하도록 설계되었기 때문
> - **정보 소비**보다 **정보 소화**가 중요하다

### 흔한 실수들
- ❌ 밑줄만 치고 끝내기 → 기억에 별 도움 안 됨
- ❌ 모든 내용을 요약하려고 함 → 핵심을 놓침
- ❌ 읽는 데만 시간 투자 → 소화할 시간이 없음
- ❌ 전자책으로만 읽기 → 감각적 지표 부족

---

## 💡 핵심 원칙: 소비 vs 소화

```
📥 소비 (정보 입력)          📤 소화 (정보 출력)
   읽기, 듣기, 보기     →     쓰기, 말하기, 적용하기
   
   시간 비율: 30%              시간 비율: 70%
```

> [!tip] 황금 비율
> 정보를 읽는 시간을 줄이고, **적용하거나 연습하는 소화 시간**에 더 많은 시간을 투자하라!

---

## 🎯 실전 독서법 체계

### 1️⃣ 읽기 전 (Before Reading)

#### Coleridge 효과 활용
- [ ] **내가 해결하려는 문제/질문을 먼저 적는다**
- [ ] 목차를 보며 호기심 유발 → 궁금한 점 적기
- [ ] 이 책에서 얻고 싶은 것 1~3개 정하기

```
📋 독서 전 체크리스트
┌─────────────────────────────────────┐
│ 🎯 내 문제/질문:                      │
│                                     │
│ ❓ 목차에서 궁금한 것들:               │
│                                     │
│ ✨ 이 책에서 얻고 싶은 것:             │
│                                     │
└─────────────────────────────────────┘
```

#### 종이책 권장
- 종이책은 **다감각적 경험** 제공
  - 표지 색감, 무게감, 두께, 위치 감각, 종이 질감
- 이런 지표들이 **기억 인출의 인덱스** 역할
- 전자책보다 **불러오기(Recall)** 가 훨씬 쉬움

---

### 2️⃣ 읽는 중 (During Reading)

#### 🔑 핵심 키워드 메모법 (김익한 교수)

> [!warning] 메모의 핵심 원칙
> **많이 하지 마라!** 핵심 키워드 2~3개만 쓰려고 결심하라.

1. **액기스만 찾기**: 1개 장을 요약할 때 **반쪽 넘지 않게**
2. **키워드에 동그라미**: 문장이 아닌 **핵심 단어**에 표시
3. **책을 보지 않고 메모**: "순간의 생각"으로 적기

```
📝 메모 예시
┌─────────────────────────────────────────────────┐
│ ❌ 잘못된 예: 문장 전체를 그대로 필사            │
│                                                 │
│ ✅ 좋은 예: "인지적 경제성" ← 핵심 키워드만       │
│            "범주화" ← 나중에 이걸 보고 내용 떠올림│
└─────────────────────────────────────────────────┘
```

#### 순간의 생각과 이음
1. 두 페이지 읽고 머리 들기
2. 책을 보지 않고 "이게 핵심"이라 생각하는 것 메모
3. 5개쯤 쌓이면 "생각의 이음" → 흐름 정리

#### 적극적 소통
- 마음에 안 드는 내용 옆에 "왜?" 적기
- 재미있는 부분에 "ㅋㅋ" 넣기
- 궁금한 것 바로바로 적어두기
- 반박하고 싶은 내용에 질문 던지기

---

### 3️⃣ 읽은 후 (After Reading)

#### 📊 구조화 (논픽션)

**목차 활용 전체 구조 파악**
1. 서론 + 목차로 책 전체 그림 그리기
2. 각 챕터 내 세부 구조 파악
3. **그림(구조도)으로 시각화**

```
📐 구조화 예시

          ┌─ 1-7장: 현상 분석
책 전체 ──┤
          └─ 8-10장: 원인 분석
                    │
                    ├─ 외생적 원인 A
                    ├─ 외생적 원인 B
                    └─ 외생적 원인 C
```

**구조화 방법**
- 마인드맵 그리기
- 상하위 개념 목록 만들기
- 인과관계 화살표로 표시
- 디지털 노트에서 작은 노트들을 묶는 큰 노트 만들기

#### ✍️ 출력하기 (가장 중요!)

> [!caution] 핵심 통찰
> **진짜 시간 낭비는 책을 읽고 나서 머리에 하나도 안 남는 것이다!**

**10줄 에세이 작성법**
1. 책 핵심 내용 **5줄 요약**
2. **내 생각 5줄** 덧붙이기
3. 블로그나 옵시디언에 기록

**출력의 효과**
- 지식의 재생산 과정
- 기존 지식 + 새 지식 = 더 깊은 통찰
- **반영구적으로 자신의 것**이 됨

## 📚 정보 유형별 소화 전략 (PACER)

| 유형 | 정의 | 소화 전략 | 예시 |
|------|------|-----------|------|
| **P**rocedural (절차형) | 어떻게 하는지 | **연습 (Practice)** | 코딩, 언어, 기술 |
| **A**nalogous (유사형) | 비유/연결 | **비판 (Critic)** | 전류=물의 흐름 |
| **C**onceptual (개념형) | 이것은 무엇인가 | **구조화 (Structure)** | 이론, 원리, 사실 |
| **E**vidence (근거형) | 구체적 증거 | **수집+반복** | 통계, 사례 |
| **R**eference (참고형) | 세부 데이터 | **수집+간격반복** | 날짜, 수치 |

### 각 유형별 실천법

#### P - 절차형
- **즉시 실전 적용** 아니면 금세 잊혀짐
- 연습할 시간 없으면 읽기도 멈춰라
- 할 일 목록에 "연습하기" 추가

#### A - 유사형
- "내가 아는 것과 어떻게 연결되나?"
- 비유가 적절한지 **비판적으로 검토**
- 더 나은 비유 찾아보기

#### C - 개념형
- **마인드맵** 그리기
- 상하위 관계 정리
- 전체 구조 재구성

#### E & R - 근거/참고형
- 일단 **수집** (속도가 핵심)
- 나중에 **간격 반복**으로 암기
- 플래시카드 활용

---

## 🔁 복습과 반복

### 간격 반복 (Spaced Repetition)
```
1일 → 3일 → 7일 → 14일 → 30일 → 90일
```
- 암기 카드 활용
- 매일 30분 정해진 시간에 복습
- 옵시디언 Spaced Repetition 플러그인 활용

### 능동적 회상 (Active Recall)
- 책 덮고 기억나는 것 적기
- 키워드 보고 내용 떠올리기
- "이게 뭐였지?" 질문 던지기

### 관심 분야 반복 독서
- 같은 주제 여러 권 읽기
- 반복되는 내용 = 핵심
- 다양한 관점으로 **입체적 이해**

## ⭐ 유명 독서법 모음

### 1. SQ3R 독서법
| 단계 | 내용 |
|------|------|
| **S**urvey | 훑어보기 - 목차, 제목, 그림 |
| **Q**uestion | 질문 만들기 |
| **R**ead | 읽기 - 답을 찾으며 |
| **R**ecite | 암송 - 책 덮고 말하기 |
| **R**eview | 복습 |

### 2. 페인만 학습법
1. 개념 선택 후 종이에 제목 쓰기
2. **12살에게 설명하듯** 쓰기
3. 막히는 부분 = 이해 안 된 부분 → 다시 공부
4. 반복하여 완전히 이해될 때까지

### 3. 코넬 노트법
```
┌─────────────────────────────────────┐
│    단서(Cue)    │    노트(Note)     │
│    20%         │    80%            │
│                │                   │
│  키워드        │  수업/독서 내용    │
│  질문          │  상세 노트         │
│                │                   │
├────────────────┴───────────────────┤
│           요약 (Summary)            │
│         핵심 내용 3줄 정리           │
└─────────────────────────────────────┘
```

### 4. 마진 리딩 (Marginalia)
- 책 여백에 적극적으로 메모
- 질문, 반박, 연결, 감정 모두 기록
- 나만의 대화 흔적 남기기

## 🛠️ 나의 실천 계획

### 📖 독서 전
- [ ] 해결하고 싶은 문제/질문 1개 적기
- [ ] 목차 보며 궁금한 점 3개 적기
- [ ] 가능하면 종이책으로 읽기

### 📝 독서 중
- [ ] 핵심 키워드 2~3개만 메모 (1장당 반쪽 이내)
- [ ] 키워드에 동그라미, 문장 말고!
- [ ] 2페이지마다 "순간의 생각" 메모
- [ ] 질문, 반박, 연결 적극적으로 적기

### ✍️ 독서 후
- [ ] 목차 보며 전체 구조도 그리기
- [ ] 10줄 에세이 작성 (요약 5줄 + 생각 5줄)
- [ ] 핵심 키워드로 플래시카드 만들기
- [ ] 1주일 후 복습

### 🔁 복습
- [ ] 간격 반복 스케줄 지키기
- [ ] 관심 분야는 비슷한 책 여러 권 읽기
- [ ] 누군가에게 배운 내용 설명해보기

## ❓ 마음가짐

> [!note] 조바심 버리기
> - 책 한 권에서 **한 가지 깨달음**만 얻어도 충분
> - 모든 내용을 다 기억할 필요 없음
> - 읽었을 때 재밌으면 그것만으로도 가치 있음
> - 픽션은 기억 못 해도 괜찮음 - 경험이 쌓이는 것

## 📝 FC
#flashcards/self

효과적인 독서의 황금 비율:: 소비 30% : 소화 70% - 읽는 시간보다 적용/출력하는 시간이 더 중요하다

핵심 키워드 메모법의 원칙:: 많이 쓰지 말고 액기스만! 1장당 반쪽 이내, 키워드 2~3개만

PACER 소화 전략:: P(절차형)→연습, A(유사형)→비판, C(개념형)→구조화, E(근거형)/R(참고형)→수집+반복

SQ3R 독서법:: Survey(훑어보기)→Question(질문)→Read(읽기)→Recite(암송)→Review(복습)

종이책이 기억에 좋은 이유:: 다감각적 경험(무게, 질감, 위치 등)이 기억 인출의 인덱스 역할

10줄 에세이 작성법:: 책 핵심 5줄 요약 + 내 생각 5줄 = 지식의 자기화
