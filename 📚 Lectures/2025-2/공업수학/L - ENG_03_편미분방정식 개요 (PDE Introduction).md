# 편미분방정식 개요 (PDE Introduction)

## 1. 편미분방정식이란?

**편미분방정식(Partial Differential Equation, PDE)**은 두 개 이상의 독립변수에 대한 편미분을 포함하는 방정식이다.

### 왜 필요한가?

자연 세계의 현상들 - 진동하는 줄, 전파되는 음파, 막대를 따라 퍼지는 열 - 은 **시간과 공간에 따라 변하는 양**을 다룬다.

- 진동하는 현의 모든 점에서의 움직임을 기술
- 막대의 각 위치에서 온도가 시간에 따라 어떻게 변하는지 예측
- 파동이 공간을 통해 어떻게 전파되는지 계산

---

## 2. PDE의 분류

### 차원별 분류

| 차원 | 방정식 종류 | 미지함수 |
|------|------------|----------|
| 1차원 | 파동방정식, 열전도방정식 | $u(x,t)$ |
| 2차원 | 라플라스방정식, 포아송방정식, 파동방정식 | $u(x,y,t)$ |
| 3차원 | 라플라스방정식, 열전도방정식 | $u(x,y,z,t)$ |

### 2계 선형 PDE 표준형

$$Au_{xx} + 2Bu_{xy} + Cu_{yy} = F(x, y, u, u_x, u_y)$$

### 판별식에 의한 분류

**판별식**: $\Delta = B^2 - AC$

| 조건 | 유형 | 대표 방정식 |
|------|------|------------|
| $\Delta > 0$ | **쌍곡형 (Hyperbolic)** | 파동방정식 |
| $\Delta = 0$ | **포물형 (Parabolic)** | 열전도방정식 |
| $\Delta < 0$ | **타원형 (Elliptic)** | 라플라스방정식 |

## 3. 주요 편미분방정식

### 파동방정식 (Wave Equation)

$$\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}$$

또는 $u_{tt} = c^2 u_{xx}$

**물리적 의미**: 
- 좌변: "가속도"
- 우변: "곡률 × c²"
- 등호: 곡률이 크면 빠르게 변한다

**비유**: 줄넘기 줄의 움직임 - 각 점의 가속도는 그 점 주변의 곡률에 비례

### 열전도방정식 (Heat Equation)

$$\frac{\partial u}{\partial t} = c^2 \frac{\partial^2 u}{\partial x^2}$$

또는 $u_t = c^2 u_{xx}$

**물리적 의미**: 온도의 시간 변화율은 공간적 온도 분포의 곡률에 비례

**비유**: 뜨거운 커피가 식는 과정 - 가장자리는 빨리 식고, 중심부는 천천히 식음

### 라플라스 방정식 (Laplace's Equation)

$$\nabla^2 u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$

**특징**: 정상 상태(steady state)를 기술

## 4. 경계조건과 초기조건

### 경계조건 (Boundary Conditions)

공간적 경계에서의 조건

**예시**: 양 끝이 고정된 현
$$u(0, t) = 0, \quad u(L, t) = 0 \quad \text{for all } t \ge 0$$

**유형**:
- **Dirichlet 조건**: 함수값 지정 ($u = f$)
- **Neumann 조건**: 도함수 지정 ($\frac{\partial u}{\partial n} = g$)
- **Robin 조건**: 혼합 ($\alpha u + \beta \frac{\partial u}{\partial n} = g$)

### 초기조건 (Initial Conditions)

$t=0$에서의 조건

**예시**: 진동하는 현
- 초기 변위: $u(x, 0) = f(x)$
- 초기 속도: $u_t(x, 0) = g(x)$

---

## 5. 해법 개요

### 변수 분리법 (Separation of Variables)

1. 해를 $u(x,t) = X(x)T(t)$ 형태로 가정
2. PDE를 두 개의 ODE로 분리
3. 경계조건으로 고유값 문제 해결
4. 중첩 원리로 일반해 구성
5. 초기조건으로 계수 결정

**적합한 경우**: 유한 영역, 동차 경계조건

### 특성방정식 방법 (Method of Characteristics)

1. 특성방정식으로 좌표변환 찾기
2. 새 좌표에서 PDE 단순화
3. 직접 적분
4. 초기조건 적용

**적합한 경우**: 무한 영역, 쌍곡형 방정식

## 6. 문제 접근 전략

### STEP 1: 문제 분류

```
- Fourier 급수? → 주기 확인 (2π or 2L)
- Fourier 적분? → 비주기, 절대적분
- Fourier 변환? → 짝/홀 판단
- PDE? → 미분 변수 확인
```

### STEP 2: 조건 체크

```
- 경계조건: u(0,t)=?, u(L,t)=?
- 초기조건: u(x,0)=?, u_t(x,0)=?
- 대칭성: 짝함수? 홀함수?
```

### STEP 3: 해법 선택

```
- 유한 영역 + 경계조건 → 변수 분리법
- 무한 영역 → 달랑베르 해법 / 푸리에 변환
```
