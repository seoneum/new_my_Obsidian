# 양화논리 기초 (Quantificational Logic)

## 개요
양화논리(술어논리)는 정언논리의 한계를 극복하고, **내부 구조**를 분석하여 더 세밀한 추론을 가능하게 한다.

---

## 1. 기본 용어

### 지시적 표현 vs 술어적 표현

| 구분 | 설명 | 예시 | 기호 |
|:---|:---|:---|:---|
| **지시적 표현 (상항)** | 특정 대상을 가리킴 | 철수, 서울, 3 | a, b, c, ... |
| **술어적 표현** | 대상의 성질/관계 표현 | ~은 학생이다, ~은 ~보다 크다 | F, G, H, ... |
| **변항** | 대상의 빈자리 (아직 미정) | x, y, z | x, y, z |

### 예시 분석
```
"철수는 학생이다"
→ 철수: 상항 (a)
→ ~은 학생이다: 1항 술어 (Sx)
→ 기호화: Sa

"3은 5보다 작다"
→ 3: 상항 (a), 5: 상항 (b)
→ ~은 ~보다 작다: 2항 술어 (Lxy)
→ 기호화: Lab
```

## 2. 열린 문장과 닫힌 문장

| 구분 | 정의 | 진리치 | 예시 |
|:---|:---|:---|:---|
| **열린 문장 (Open Sentence)** | 자유변항이 있는 문장 | 없음 | Fx, Gxy |
| **닫힌 문장 (Closed Sentence)** | 자유변항이 없는 문장 | 있음 | Fa, ∀xFx |

> **자유변항**: 양화사에 묶이지 않은 변항
> **속박변항**: 양화사에 묶인 변항

### 예시
```
∀x(Fx → Gx)
   ↑     ↑
   속박   속박  → 닫힌 문장 (진리치 있음)

Fx → ∀yGy
↑
자유         → 열린 문장 (진리치 없음)
```

## 3. 양화사 (Quantifiers)

### 보편양화사 (Universal Quantifier): ∀

| 기호 | 읽기 | 의미 |
|:---:|:---|:---|
| ∀x | 모든 x에 대해 | 논의영역의 **모든** 대상이 만족 |

### 존재양화사 (Existential Quantifier): ∃

| 기호 | 읽기 | 의미 |
|:---:|:---|:---|
| ∃x | 어떤 x가 존재하여 | 논의영역의 **적어도 하나**가 만족 |

### 양화사 간의 관계

```
∀xFx ≡ ¬∃x¬Fx    (모두 F ↔ F 아닌 것 없음)
∃xFx ≡ ¬∀x¬Fx    (어떤 것 F ↔ 모두 F 아닌 것 아님)

¬∀xFx ≡ ∃x¬Fx    (모두 F 아님 ↔ F 아닌 것 존재)
¬∃xFx ≡ ∀x¬Fx    (어떤 것도 F 아님 ↔ 모두 F 아님)
```

## 4. 정언문장의 양화논리 기호화

### A, E, I, O 문장 변환

| 유형 | 한국어 | 기호화 | 주연결사 |
|:---:|:---|:---|:---:|
| **A** | 모든 F는 G이다 | ∀x(Fx → Gx) | → |
| **E** | 어떤 F도 G가 아니다 | ∀x(Fx → ¬Gx) | → |
| **I** | 어떤 F는 G이다 | ∃x(Fx ∧ Gx) | ∧ |
| **O** | 어떤 F는 G가 아니다 | ∃x(Fx ∧ ¬Gx) | ∧ |

### 중요한 차이점

**∀ (전칭)과 조건문 (→)**
- "모든 고양이는 동물이다" = ∀x(Cx → Ax)
- ∀x(Cx ∧ Ax)로 쓰면 **오류** (모든 것이 고양이이면서 동물?)

**∃ (존재)와 연언 (∧)**
- "어떤 고양이는 검다" = ∃x(Cx ∧ Bx)
- ∃x(Cx → Bx)로 쓰면 **약해짐** (고양이 아닌 것도 만족)

### 기억법
```
∀ → 조건문 (→): "모든 F가 G"라고 해서 모든 게 F인 건 아님
∃ → 연언 (∧): "어떤 F가 G"면 그것은 F이면서 G여야 함
```

## 5. 복합 문장의 기호화

### 예시 1: "학생만 시험을 본다"
```
오직 학생만 시험을 본다
= 시험을 보면 학생이다
= ∀x(Ex → Sx)
```

### 예시 2: "모든 학생이 모든 시험에 합격했다"
```
∀x(Sx → ∀y(Ey → Pxy))
또는 ∀x∀y((Sx ∧ Ey) → Pxy)
```

### 예시 3: "어떤 학생은 어떤 시험도 안 봤다"
```
∃x(Sx ∧ ∀y(Ey → ¬Txy))
= 어떤 학생 x가 있어서, 모든 시험 y에 대해, x는 y를 보지 않았다
```

### 예시 4: "아무도 모든 것을 알지 못한다"
```
¬∃x∀yKxy
= ∀x¬∀yKxy
= ∀x∃y¬Kxy
= 모든 사람은 모르는 것이 있다
```

## 6. 양화사 전환 규칙 (Quantifier Negation)

### 기본 동치 관계

| 원래 형태 | 동치 형태 |
|:---|:---|
| ¬∀xφ | ∃x¬φ |
| ¬∃xφ | ∀x¬φ |
| ∀xφ | ¬∃x¬φ |
| ∃xφ | ¬∀x¬φ |

### 직관적 이해
```
¬∀xFx = "모두 F인 건 아니다" = "F 아닌 게 있다" = ∃x¬Fx
¬∃xFx = "F인 게 없다" = "모두 F 아니다" = ∀x¬Fx
```

## 7. 논의영역 (Domain of Discourse)

양화사의 범위를 제한하는 집합

### 예시
논의영역 D = {1, 2, 3}일 때:

```
∀xFx ≡ Fa ∧ Fb ∧ Fc  (1, 2, 3 모두 F)
∃xFx ≡ Fa ∨ Fb ∨ Fc  (1, 2, 3 중 하나 이상 F)
```

### 진리치 판정
논의영역 D = {a, b}, 해석 I: Fa = T, Fb = F, Ga = F, Gb = T

| 문장 | 계산 | 진리치 |
|:---|:---|:---:|
| ∀xFx | Fa ∧ Fb = T ∧ F | **F** |
| ∃xFx | Fa ∨ Fb = T ∨ F | **T** |
| ∀x(Fx → Gx) | (Fa→Ga) ∧ (Fb→Gb) = (T→F) ∧ (F→T) = F ∧ T | **F** |
| ∃x(Fx ∧ Gx) | (Fa∧Ga) ∨ (Fb∧Gb) = (T∧F) ∨ (F∧T) = F ∨ F | **F** |

## 8. 타당성과 반례

### 양화논리에서의 타당성
- **타당한 논증**: 어떤 해석에서도 전제가 참이면 결론도 참
- **반례**: 전제는 참이지만 결론이 거짓인 해석

### 반례 찾기 예시
```
논증: ∃xFx, ∃xGx ⊢ ∃x(Fx ∧ Gx)

반례:
논의영역 D = {a, b}
해석: Fa = T, Fb = F, Ga = F, Gb = T

전제1: ∃xFx = T (a가 F)
전제2: ∃xGx = T (b가 G)
결론: ∃x(Fx ∧ Gx) = F (F이면서 G인 것 없음)

→ 부당한 논증!
```

## 9. 정언논리 vs 양화논리

| 측면 | 정언논리 | 양화논리 |
|:---|:---|:---|
| 분석 단위 | 문장 전체 | 문장 내부 구조 |
| 표현력 | A, E, I, O만 | 무한한 복합문 |
| 다중 양화 | 불가 | 가능 |
| 관계 표현 | 제한적 | 자유로움 |

---

## 핵심 정리

1. **상항 vs 술어 vs 변항**: 이름(a,b,c) vs 성질(F,G) vs 빈자리(x,y,z)
2. **열린/닫힌 문장**: 자유변항 유무에 따라 진리치 결정
3. **양화사**: ∀(모든), ∃(어떤)
4. **기호화 공식**:
   - A: ∀x(Fx → Gx)
   - E: ∀x(Fx → ¬Gx)
   - I: ∃x(Fx ∧ Gx)
   - O: ∃x(Fx ∧ ¬Gx)
5. **양화사 전환**: ¬∀x = ∃x¬, ¬∃x = ∀x¬
