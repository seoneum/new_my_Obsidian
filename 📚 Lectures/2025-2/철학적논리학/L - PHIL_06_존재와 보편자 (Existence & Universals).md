# 존재와 보편자 (Existence & Universals)

## 개요
논리학의 형식적 도구를 철학적 문제에 적용하여, **존재의 의미**와 **보편자의 지위**를 분석한다.

---

## 1. "존재한다"는 술어인가?

### 문제 제기
```
"소크라테스는 존재한다" (?)
"페가수스는 존재하지 않는다" (?)
```

존재가 술어라면:
- Ex (x는 존재한다)로 표현 가능
- 그러나 "페가수스는 존재하지 않는다"는 어떻게?
- 존재하지 않는 것에 대해 말할 수 있는가?

---

## 2. 마이농의 입장 (Meinong)

### 존재를 술어로 봄
```
"황금산은 존재하지 않는다"
→ 황금산이라는 대상이 있고, 그것은 존재 속성이 없다
→ 존재와 존립(subsistence)의 구분
```

### 마이농의 존재론적 정글
- **존재 (Existence)**: 시공간적 대상
- **존립 (Subsistence)**: 추상적 대상 (수, 명제)
- **비존재 대상**: 황금산, 둥근 사각형

### 문제점
- 존재론적으로 과잉 (너무 많은 것이 "있다")
- "둥근 사각형은 둥글면서 사각이다"를 인정해야 함

---

## 3. 프레게-러셀의 입장

### 존재는 술어가 아니라 양화사

**프레게 (Frege)**
```
"존재한다" = "개념 아래에 대상이 있다"
존재는 1차 술어가 아니라 2차 술어 (개념에 대한 술어)

"말이 존재한다" = "말이다라는 개념 아래 뭔가가 있다"
                = ∃xHx
```

**러셀 (Russell)**
```
"페가수스는 존재하지 않는다"
= "비행하는 말이라는 속성을 가진 것은 없다"
= ¬∃x(비행하는-말(x))
```

### 확정기술구 이론 (Russell's Theory of Definite Descriptions)
```
"현재 프랑스 왕은 대머리다"

분석:
1. 프랑스 왕이 존재한다 (∃x...)
2. 오직 하나다 (∀y... → y=x)
3. 그것은 대머리다 (...∧ Bx)

= ∃x(Kx ∧ ∀y(Ky → y=x) ∧ Bx)
```

### 장점
- 비존재 대상에 대한 주장을 의미있게 분석
- 존재론적 절약 (Ontological Parsimony)

---

## 4. 존재론적 개입 (Ontological Commitment)

### 콰인의 기준
```
"존재한다는 것은 변항의 값이 되는 것이다"
"To be is to be the value of a bound variable"
```

### 세 가지 개입 방식

| 방식 | 표현 | 개입 |
|:---|:---|:---|
| **단칭어** | Fa (a는 F다) | a가 존재한다고 개입 |
| **양화** | ∃xFx | F인 것이 존재한다고 개입 |
| **동일성** | a = b | a와 b가 존재한다고 개입 |

### 예시
```
"수는 존재한다"를 주장하려면?
→ ∃x(x는 수다)를 참으로 받아들이면 됨
→ 수에 대한 존재론적 개입
```

---

## 5. 보편자 논쟁 (Problem of Universals)

### 문제
```
"빨강"은 존재하는가?
"빨간 사과", "빨간 자동차"에서 공통된 "빨강"은 무엇인가?
```

### 두 입장

| 입장 | 주장 | 대표자 |
|:---|:---|:---|
| **실재론** | 보편자는 존재한다 | 플라톤, 러셀 |
| **유명론** | 보편자는 이름일 뿐 | 오컴, 콰인 |

---

## 6. 실재론 (Realism)

### 플라톤적 실재론
```
- 보편자는 개별자와 독립적으로 존재
- 이데아계에 "빨강 그 자체"가 있음
- 개별적 빨간 것들은 "빨강"에 분유(participation)
```

### 온건한 실재론 (아리스토텔레스)
```
- 보편자는 개별자 안에 내재
- "빨강"은 빨간 것들 안에만 존재
- 독립적 존재가 아님
```

### 실재론의 장점
- 속성 공유 설명 용이
- 수학적 대상의 객관성 확보
- 법칙의 보편성 설명

---

## 7. 유명론 (Nominalism)

### 기본 주장
```
- 존재하는 것은 개별자뿐
- "빨강"은 빨간 개별자들의 집합 또는 이름
- 보편자에 대한 존재론적 개입 불필요
```

### 유명론의 종류

| 유형 | 설명 |
|:---|:---|
| **술어 유명론** | "빨강"은 술어일 뿐, 대상 아님 |
| **집합 유명론** | "빨강" = 빨간 것들의 집합 |
| **유사성 유명론** | 빨간 것들은 서로 유사할 뿐 |
| **트로프 이론** | 개별 속성(트로프)들의 유사성 |

### 유명론의 문제
- 유사성 자체도 보편자 아닌가?
- 수학적 대상을 어떻게 설명?

---

## 8. 굿맨의 귀납의 새로운 수수께끼

### 초랑 (Grue) 역설

**정의**
```
x는 초랑(grue)이다 ↔ 
  (t 이전에 관찰되고 초록) 또는 (t 이후에 관찰되고 파랑)
```

**문제**
```
관찰: 지금까지 본 에메랄드는 모두 초록이다
귀납1: 모든 에메랄드는 초록이다 (귀납)
귀납2: 모든 에메랄드는 초랑이다 (귀납?)

두 귀납 모두 현재 증거와 양립!
t 이후 에메랄드가 파랑이면 "초랑" 귀납이 맞음
```

### 투사 가능성 (Projectibility)
```
문제: 어떤 술어가 귀납에 적합한가?
- "초록"은 투사 가능? "초랑"은?
- 단순히 관찰 증거로는 구분 불가
```

### 굿맨의 해결 시도
```
- 정착된(entrenched) 술어: 과거에 성공적으로 사용된 술어
- "초록"은 정착, "초랑"은 비정착
- 언어 공동체의 관행에 의존
```

### 철학적 함의
- 귀납의 정당화 문제 심화
- 자연종(Natural Kinds)의 문제
- 술어의 선택이 이론 의존적

---

## 9. 자연종과 본질

### 자연종 (Natural Kinds)
```
"물", "금", "호랑이" - 자연에 실재하는 종류?
vs
"초랑", "게임", "잡동사니" - 인위적 분류?
```

### 크립키-퍼트남의 견해
```
- 자연종 용어는 고정 지시어
- "물 = H₂O"는 필연적 참
- 자연종의 본질은 발견되는 것
```

### 반론
```
- 과학 이론은 변할 수 있다
- "금"의 정의가 바뀌면?
- 생물 종의 본질은 무엇인가?
```

## 10. 요약: 존재론적 입장들

| 문제 | 실재론 | 유명론 |
|:---|:---|:---|
| 보편자 | 존재함 | 이름뿐 |
| 수학적 대상 | 존재함 | 허구/도구 |
| 속성 공유 | 동일 보편자 분유 | 유사성 |
| 법칙 | 보편자 간 관계 | 규칙성 기술 |

---

## 핵심 정리

1. **마이농**: 존재는 술어, 비존재 대상도 "있다"
2. **프레게-러셀**: 존재는 양화사, "있다" = ∃x
3. **존재론적 개입**: 변항의 값이 되는 것이 존재
4. **실재론**: 보편자는 실재, 개별자와 독립 또는 내재
5. **유명론**: 보편자는 이름, 개별자만 존재
6. **굿맨**: 귀납의 술어 선택 문제, 투사 가능성
7. **자연종**: 자연에 실재하는 종류 vs 인위적 분류
