# 관계술어와 다중양화 (Relations & Multiple Quantifiers)

## 개요
관계술어는 **둘 이상의 대상 사이의 관계**를 표현하며, 다중양화는 여러 양화사가 결합된 복잡한 문장을 다룬다.

---

## 1. n항 술어 (n-place Predicates)

### 술어의 항수

| 항수 | 의미 | 예시 | 기호 |
|:---:|:---|:---|:---|
| **1항** | 한 대상의 성질 | ~은 빨갛다 | Fx |
| **2항** | 두 대상의 관계 | ~은 ~를 사랑한다 | Lxy |
| **3항** | 세 대상의 관계 | ~은 ~에게 ~을 주다 | Gxyz |

### 순서의 중요성
```
"철수가 영희를 사랑한다" ≠ "영희가 철수를 사랑한다"

Lab ≠ Lba (a=철수, b=영희)
```

> 관계술어에서 **인자의 순서**는 의미를 결정한다

## 2. 관계의 논리적 특성

### 2.1 대칭성 (Symmetry)

| 특성 | 정의 | 예시 |
|:---|:---|:---|
| **대칭적** | Rxy → Ryx | 결혼하다, 닮다 |
| **반대칭적** | Rxy → ¬Ryx | ~보다 크다, ~의 부모 |
| **비대칭적** | 대칭도 반대칭도 아님 | 사랑하다 |

```
대칭: ∀x∀y(Rxy → Ryx)
반대칭: ∀x∀y(Rxy → ¬Ryx)
```

### 2.2 재귀성 (Reflexivity)

| 특성 | 정의 | 예시 |
|:---|:---|:---|
| **재귀적** | ∀xRxx | 동일하다, ~이상이다 |
| **반재귀적** | ∀x¬Rxx | ~보다 크다, ~의 부모 |
| **비재귀적** | 재귀도 반재귀도 아님 | 사랑하다 |

```
재귀: ∀xRxx (자기 자신과도 성립)
반재귀: ∀x¬Rxx (자기 자신과는 절대 불성립)
```

### 2.3 이행성 (Transitivity)

| 특성 | 정의 | 예시 |
|:---|:---|:---|
| **이행적** | (Rxy ∧ Ryz) → Rxz | ~보다 크다, 조상이다 |
| **반이행적** | (Rxy ∧ Ryz) → ¬Rxz | 아버지이다 |
| **비이행적** | 이행도 반이행도 아님 | 친구이다 |

```
이행: ∀x∀y∀z((Rxy ∧ Ryz) → Rxz)
반이행: ∀x∀y∀z((Rxy ∧ Ryz) → ¬Rxz)
```

## 3. 특별한 관계 유형

### 3.1 동등관계 (Equivalence Relation)
**재귀적 + 대칭적 + 이행적**

```
예: 동일한 나이, 같은 학년, 합동(기하학)

조건:
1. ∀xRxx                    (재귀)
2. ∀x∀y(Rxy → Ryx)          (대칭)
3. ∀x∀y∀z((Rxy ∧ Ryz) → Rxz) (이행)
```

### 3.2 순서관계 (Ordering Relation)

**부분순서 (Partial Order)**: 재귀적 + 반대칭적 + 이행적
```
예: ≤ (이하), 집합의 포함관계 ⊆
```

**전순서 (Total Order)**: 부분순서 + 완전성
```
예: 실수의 ≤ (임의의 두 수 비교 가능)
완전성: ∀x∀y(Rxy ∨ Ryx)
```

**엄격순서 (Strict Order)**: 반재귀적 + 반대칭적 + 이행적
```
예: < (미만), 진부분집합 ⊂
```

## 4. 동일성과 유사성

### 동일성 (Identity): =

| 특징 | 설명 |
|:---|:---|
| 정의 | 정확히 같은 하나의 대상 |
| 기호 | x = y |
| 특성 | 재귀, 대칭, 이행 (동등관계) |

### 대입 원리 (Leibniz's Law)
```
a = b → (Fa ↔ Fb)
동일한 대상은 모든 성질을 공유한다
```

### 유사성 vs 동일성
```
"저 두 자동차는 같다" 
→ 동일성: 하나의 자동차 (불가능한 해석)
→ 유사성: 같은 모델, 같은 색 등 (가능한 해석)

논리적 동일성(=)은 수적 동일성만 표현
```

## 5. 동일성을 이용한 표현

### "정확히 하나 존재" (Exactly One)
```
∃x(Fx ∧ ∀y(Fy → y = x))

"F인 x가 존재하고, F인 것은 모두 그 x와 동일하다"
```

### "정확히 둘 존재" (Exactly Two)
```
∃x∃y(Fx ∧ Fy ∧ x ≠ y ∧ ∀z(Fz → (z = x ∨ z = y)))
```

### "최소 둘 존재" (At Least Two)
```
∃x∃y(Fx ∧ Fy ∧ x ≠ y)
```

### "최대 하나 존재" (At Most One)
```
∀x∀y((Fx ∧ Fy) → x = y)
```

## 6. 다중양화 (Multiple Quantification)

### 양화사 순서의 중요성

**순서가 같을 때 (동종 양화사)**
```
∀x∀yRxy ≡ ∀y∀xRxy (순서 무관)
∃x∃yRxy ≡ ∃y∃xRxy (순서 무관)
```

**순서가 다를 때 (이종 양화사)**
```
∀x∃yRxy ≢ ∃y∀xRxy (순서 중요!)
```

### 예시: "모든 사람은 누군가를 사랑한다"

```
∀x∃yLxy
= 모든 x에 대해, 어떤 y가 있어서, x가 y를 사랑한다
= 각자 사랑하는 사람이 (다를 수) 있다
```

### 예시: "누군가가 모든 사람에게 사랑받는다"

```
∃y∀xLxy
= 어떤 y가 있어서, 모든 x가 y를 사랑한다
= 한 사람이 모두에게 사랑받는다
```

### 관계 정리

| 문장 | 의미 | 강도 |
|:---|:---|:---:|
| ∀x∀yLxy | 모두가 모두를 사랑 | 가장 강함 |
| ∀x∃yLxy | 모두가 누군가를 사랑 | ↓ |
| ∃y∀xLxy | 누군가가 모두에게 사랑받음 | ↓ |
| ∃x∃yLxy | 누군가가 누군가를 사랑 | 가장 약함 |

### 함의 관계
```
∀x∀yLxy → ∀x∃yLxy  (T)
∀x∀yLxy → ∃y∀xLxy  (T)
∃y∀xLxy → ∃x∃yLxy  (T)
∀x∃yLxy → ∃y∀xLxy  (F) ← 핵심!
```

## 7. 해석과 진리치

### 해석 (Interpretation)
- **논의영역** (Domain): 변항이 가리킬 수 있는 대상의 집합
- **상항 지칭**: 각 상항이 가리키는 특정 대상
- **술어 외연**: 각 술어를 만족하는 대상(순서쌍)의 집합

### 진리치 계산 예시

**문장**: ∀x∃yLxy
**해석**: D = {a, b}, Lxy = "x가 y를 사랑"
**외연**: L = {(a,b), (b,a)}

```
∀x∃yLxy
= (∃yLay) ∧ (∃yLby)           [x에 a, b 대입]
= (Laa ∨ Lab) ∧ (Lba ∨ Lbb)   [y에 a, b 대입]
= (F ∨ T) ∧ (T ∨ F)           [외연 확인]
= T ∧ T
= T
```

## 8. 복잡한 기호화 연습

### "모든 사람은 자신보다 큰 수를 안다"
```
∀x(Px → ∃y(Ny ∧ Gyx ∧ Kxy))
P: 사람, N: 수, G: ~보다 크다, K: 알다
```

### "어떤 사람도 모든 것을 알지 못한다"
```
¬∃x(Px ∧ ∀yKxy)
= ∀x(Px → ∃y¬Kxy)
= 모든 사람은 모르는 것이 있다
```

### "누군가를 모두가 존경하는 사람이 있다"
```
∃x(Px ∧ ∃y(Py ∧ ∀z(Pz → Rzy)))
= 어떤 사람 x가 있는데, 어떤 사람 y가 있어서, 
  모든 사람 z가 y를 존경하고, x는 그런 사람이다

또는 더 간단히:
∃y(Py ∧ ∀z(Pz → Rzy))
```

## 9. 흔한 오류와 주의점

### 양화사 순서 혼동
```
잘못: "모든 사람에게 그를 사랑하는 사람이 있다"를
      ∃y∀xLyx로 기호화 (한 사람이 모두를 사랑)

맞음: ∀x∃yLyx (각 사람마다 다른 사람이 사랑할 수 있음)
```

### 변항 혼동
```
잘못: ∀x∃xLxx (같은 변항 중복 사용)
맞음: ∀x∃yLxy (다른 변항 사용)
```

### 논의영역 무시
```
∀xFx에서 D가 공집합이면?
→ 공허하게 참 (Vacuously True)
→ 반례가 없으므로 거짓이 아님
```

## 핵심 정리

1. **n항 술어**: 순서쌍으로 관계 표현, 순서 중요
2. **관계 특성**: 대칭/반대칭, 재귀/반재귀, 이행/반이행
3. **동등관계**: 재귀 + 대칭 + 이행
4. **순서관계**: 재귀 + 반대칭 + 이행 (부분순서)
5. **동일성(=)**: 수적 동일성, 대입 원리
6. **다중양화**: ∀x∃y ≢ ∃y∀x (순서 중요!)
7. **해석**: 논의영역 + 상항지칭 + 술어외연
