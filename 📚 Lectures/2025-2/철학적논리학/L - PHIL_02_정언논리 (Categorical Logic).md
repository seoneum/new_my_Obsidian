# 정언논리 (Categorical Logic)

## 개요
정언논리는 **아리스토텔레스의 전통 논리학**에서 유래한 체계로, 주어와 술어의 관계를 통해 명제를 분석한다.

---

## 1. 정언문장의 네 가지 유형 (A, E, I, O)

| 유형 | 명칭 | 형식 | 예시 | 양 | 질 |
|:---:|:---:|:---|:---|:---:|:---:|
| **A** | 전칭긍정 | 모든 S는 P이다 | 모든 사람은 동물이다 | 전칭 | 긍정 |
| **E** | 전칭부정 | 어떤 S도 P가 아니다 | 어떤 새도 포유류가 아니다 | 전칭 | 부정 |
| **I** | 특칭긍정 | 어떤 S는 P이다 | 어떤 학생은 천재이다 | 특칭 | 긍정 |
| **O** | 특칭부정 | 어떤 S는 P가 아니다 | 어떤 개는 검지 않다 | 특칭 | 부정 |

### 기억법
- **A**ffirmo (긍정한다) → A, I는 긍정
- n**E**go (부정한다) → E, O는 부정

## 2. 벤다이어그램 (Venn Diagram)

벤다이어그램은 정언문장의 **진리조건**을 시각적으로 표현한다.

### 표기 규칙

| 표기 | 의미 |
|:---:|:---|
| 빗금(음영) | 해당 영역이 **비어있음** (원소 없음) |
| X 표시 | 해당 영역에 **최소 하나의 원소** 존재 |
| 빈 영역 | **정보 없음** (있을 수도, 없을 수도) |

### 네 가지 문장의 벤다이어그램

```
A: 모든 S는 P이다           E: 어떤 S도 P가 아니다
   ┌─────┬─────┐               ┌─────┬─────┐
   │ S   │  P  │               │ S   │  P  │
   │ /// │     │               │     │ /// │
   │ /// │     │               │     │ /// │
   └─────┴─────┘               └─────┴─────┘
   S이면서 P아닌 영역 빗금      S이면서 P인 영역 빗금

I: 어떤 S는 P이다            O: 어떤 S는 P가 아니다
   ┌─────┬─────┐               ┌─────┬─────┐
   │ S   │  P  │               │ S   │  P  │
   │     │  X  │               │  X  │     │
   │     │     │               │     │     │
   └─────┴─────┘               └─────┴─────┘
   교집합에 X                   S-P 영역에 X
```

## 3. 정언적 삼단논증 (Categorical Syllogism)

### 구조
삼단논증은 **두 개의 전제**와 **하나의 결론**으로 구성된다.

```
대전제 (Major Premise): M - P의 관계
소전제 (Minor Premise): S - M의 관계
─────────────────────────────────
결론 (Conclusion): S - P의 관계
```

### 핵심 용어

| 용어 | 정의 | 위치 |
|:---|:---|:---|
| **대명사 (Major Term, P)** | 결론의 술어 | 대전제에 등장 |
| **소명사 (Minor Term, S)** | 결론의 주어 | 소전제에 등장 |
| **매개명사 (Middle Term, M)** | 두 전제를 연결 | 결론에 등장 안 함 |

### 예시
```
대전제: 모든 사람(M)은 죽는다(P)
소전제: 소크라테스(S)는 사람(M)이다
────────────────────────────────
결론:   소크라테스(S)는 죽는다(P)
```

## 4. 격 (Figure)

격은 **매개명사(M)의 위치**에 따라 결정된다.

| 격 | 대전제 | 소전제 | 형태 |
|:---:|:---:|:---:|:---:|
| **제1격** | M - P | S - M | M이 왼쪽, 오른쪽 |
| **제2격** | P - M | S - M | M이 둘 다 오른쪽 |
| **제3격** | M - P | M - S | M이 둘 다 왼쪽 |
| **제4격** | P - M | M - S | M이 오른쪽, 왼쪽 |

### 시각적 표현
```
제1격        제2격        제3격        제4격
M - P       P - M       M - P       P - M
S - M       S - M       M - S       M - S
─────       ─────       ─────       ─────
S - P       S - P       S - P       S - P
```

## 5. 식 (Mood)

식은 **전제와 결론의 문장 유형(A, E, I, O)의 조합**이다.

### 표기
```
[대전제 유형][소전제 유형][결론 유형] - 제n격
```

### 예시
- **AAA-1 (Barbara)**: 제1격에서 대전제 A, 소전제 A, 결론 A
  ```
  모든 M은 P이다 (A)
  모든 S는 M이다 (A)
  ─────────────────
  모든 S는 P이다 (A)
  ```

- **EAE-1 (Celarent)**:
  ```
  어떤 M도 P가 아니다 (E)
  모든 S는 M이다 (A)
  ─────────────────────
  어떤 S도 P가 아니다 (E)
  ```

## 6. 타당성 검사 (Validity Check)

### 벤다이어그램을 이용한 검사

**방법**:
1. 세 개의 원(S, P, M)을 겹쳐 그린다
2. 두 전제의 정보를 순서대로 표시한다
   - 전칭문장(A, E)을 먼저 그린다 (빗금)
   - 특칭문장(I, O)을 나중에 그린다 (X)
3. 결론의 정보가 자동으로 나타나면 **타당**

### 타당성 규칙 (전통적 방법)

삼단논증이 타당하려면 다음 규칙을 **모두** 만족해야 한다:

1. **매개명사 규칙**: 매개명사는 최소 한 번 주연(周延)되어야 한다
2. **대명사/소명사 규칙**: 결론에서 주연된 명사는 전제에서도 주연되어야 한다
3. **부정 전제 규칙**: 부정 전제가 하나면 결론도 부정, 둘이면 결론 불가
4. **특칭 전제 규칙**: 두 전제가 모두 특칭이면 결론 불가

### 주연 (Distribution)

| 문장 유형 | 주어 | 술어 |
|:---:|:---:|:---:|
| A (모든 S는 P) | 주연 | 비주연 |
| E (어떤 S도 P 아님) | 주연 | 주연 |
| I (어떤 S는 P) | 비주연 | 비주연 |
| O (어떤 S는 P 아님) | 비주연 | 주연 |

> 주연: 해당 명사가 지시하는 **전체 집합**에 대해 말함

## 8. 오류 유형 (Fallacies)

| 오류명 | 위반 규칙 | 예시 |
|:---|:---|:---|
| **매개명사 비주연 오류** | 규칙 1 | 매개명사가 둘 다 술어 위치 |
| **대명사 부당 주연 오류** | 규칙 2 | 전제에서 비주연인 대명사가 결론에서 주연 |
| **소명사 부당 주연 오류** | 규칙 2 | 전제에서 비주연인 소명사가 결론에서 주연 |
| **이중 부정 오류** | 규칙 3 | 두 전제가 모두 부정 |
| **부정 전제 긍정 결론 오류** | 규칙 3 | 부정 전제에서 긍정 결론 |
| **이중 특칭 오류** | 규칙 4 | 두 전제가 모두 특칭 |

## 핵심 정리

1. **정언문장**: A(전칭긍정), E(전칭부정), I(특칭긍정), O(특칭부정)
2. **벤다이어그램**: 빗금=비어있음, X=존재함
3. **삼단논증 구성**: 대전제(M-P) + 소전제(S-M) → 결론(S-P)
4. **격**: 매개명사 위치에 따른 4가지 형태
5. **식**: 문장 유형 조합 (예: AAA, EAE, AII 등)
6. **타당성**: 벤다이어그램 또는 4가지 규칙으로 검사
