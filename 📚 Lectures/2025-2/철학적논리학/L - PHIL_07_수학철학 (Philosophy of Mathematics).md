# 수학철학 (Philosophy of Mathematics)

## 개요
수학적 대상의 존재론적 지위와 수학적 지식의 본성을 논리학적 관점에서 탐구한다.

---

## 1. 기본 문제: 수 vs 숫자

### 구분

| 용어 | 의미 | 예시 |
|:---|:---|:---|
| **숫자 (Numeral)** | 수를 표기하는 기호 | "3", "III", "三" |
| **수 (Number)** | 추상적 대상 자체 | 셋이라는 수 |

```
"3", "III", "三", "three"는 모두 다른 숫자
그러나 동일한 수를 지시
```

### 핵심 질문
```
수는 존재하는가?
존재한다면 어디에, 어떻게?
우리는 수를 어떻게 아는가?
```

---

## 2. 플라톤주의 (Platonism)

### 기본 입장
```
- 수학적 대상(수, 집합, 함수 등)은 실재한다
- 시공간 밖에 독립적으로 존재
- 인간 마음과 무관하게 객관적
```

### 주요 주장

| 측면 | 플라톤주의 입장 |
|:---|:---|
| 존재론 | 추상적 대상 실재 |
| 인식론 | 수학적 직관으로 파악 |
| 의미론 | 수학 문장은 추상 대상에 대한 참/거짓 |
| 진리 | 수학적 진리는 발견되는 것 |

### 장점
- 수학의 객관성 설명
- 수학적 진리의 필연성 설명
- 수학자들의 실제 활동과 부합

### 문제점
```
인식론적 문제:
시공간 밖의 대상을 어떻게 인식하는가?
인과적 접촉 없이 지식이 가능한가?
```

---

## 3. 형식주의 (Formalism)

### 기본 입장
```
- 수학은 기호 조작 게임
- 수학적 대상은 실재하지 않음
- 의미 있는 것은 형식 체계뿐
```

### 체스 비유 (Hilbert)
```
수학 = 체스
- 수 = 체스 말
- 공리 = 게임 규칙
- 정리 = 게임에서 도달 가능한 배치
- 증명 = 합법적 수순

체스 말에 "실재"가 없듯이, 수에도 실재 불필요
```

### 힐베르트 프로그램
```
목표: 수학의 무모순성을 유한한 방법으로 증명
방법: 메타수학 (수학에 대한 수학)

1. 수학을 형식 체계로 공리화
2. 유한한 조합적 방법으로 무모순성 증명
```

### 괴델의 불완전성 정리 (1931)

**제1 불완전성 정리**
```
충분히 강한 무모순 형식 체계에는
증명도 반증도 불가능한 문장이 존재
```

**제2 불완전성 정리**
```
충분히 강한 무모순 형식 체계는
자신의 무모순성을 증명할 수 없음
```

### 형식주의의 한계
- 힐베르트 프로그램 좌절
- 수학적 의미를 설명 못함
- 왜 특정 규칙인가?

## 4. 프레게의 논리주의 (Logicism)

### 기본 입장
```
수학 = 논리학의 확장
수학적 개념은 순수 논리적 개념으로 정의 가능
수학적 진리는 논리적 진리
```

### 수의 논리적 정의

**프레게의 정의**
```
0 = 자기 자신과 동일하지 않은 것의 수
  = |{x : x ≠ x}|
  = |∅|

1 = 0과 동일한 것의 수
  = |{x : x = 0}|
  = |{0}|

2 = 0 또는 1과 동일한 것의 수
  = |{x : x = 0 ∨ x = 1}|
  = |{0, 1}|
```

### 러셀의 역설

**발견**: 프레게의 체계에 모순
```
R = {x : x ∉ x} (자기 자신을 원소로 갖지 않는 집합들의 집합)

R ∈ R인가?
- R ∈ R이면 → 정의상 R ∉ R
- R ∉ R이면 → 정의상 R ∈ R
→ 모순!
```

### 프레게의 반응
> "과학자가 맞이할 수 있는 최악의 일은, 자신의 작업이 완성될 때 그 기초가 무너지는 것이다"

## 5. 프레게의 반심리주의 (Anti-Psychologism)

### 심리주의 비판
```
심리주의: 수학/논리는 심리적 법칙
반론: 논리는 사고의 법칙이 아니라 진리의 법칙
```

### 4가지 핵심 논증

**1. 객관성 논증**
```
수학적 진리는 누가 생각하든 동일
"2+2=4"는 생각하는 사람에 의존하지 않음
심리 상태는 개인마다 다름
→ 수학 ≠ 심리
```

**2. 규범성 논증**
```
논리는 규범적: 어떻게 사고해야 하는가
심리학은 기술적: 어떻게 사고하는가
사람들이 실제로 오류를 범함
→ 논리 ≠ 심리적 사실
```

**3. 필연성 논증**
```
수학적 진리는 필연적
심리적 사실은 우연적
다른 심리 구조를 가진 존재도 가능
→ 수학 ≠ 우연적 심리 구조
```

**4. 무한성 논증**
```
수학적 대상은 무한히 많음
인간 마음은 유한
무한을 어떻게 심리적으로 파악?
→ 수학 ≠ 유한한 심리
```

### 함의
- 수학적 대상의 객관적 실재
- 제3의 영역 (물리적도 심리적도 아닌)
- 플라톤주의로의 귀결

## 6. 콰인-퍼트남 필요불가결성 논증

### 논증 구조
```
전제 1: 우리는 최선의 과학 이론을 믿어야 한다
전제 2: 수학은 최선의 과학 이론에 필요불가결하다
전제 3: 이론을 믿으면 그 이론이 개입하는 존재자를 인정해야
────────────────────────────────────────────────────
결론: 수학적 대상이 존재한다
```

### 상세 설명

**전제 1: 자연주의**
```
철학은 과학과 연속적
최선의 과학 = 최선의 존재론 가이드
```

**전제 2: 필요불가결성**
```
물리학 공식: F = ma, E = mc²
경제학 모델: 미분방정식
생물학: 통계, 확률

수학 없이는 과학 불가능
```

**전제 3: 존재론적 개입**
```
"전자가 존재한다"를 믿는 이유?
→ 최선의 물리 이론이 전자에 개입

"수가 존재한다"도 마찬가지
→ 최선의 과학 이론이 수에 개입
```

### 결론의 의미
```
수학적 플라톤주의의 과학적 정당화
경험적 근거에 기반한 수학적 실재론
```

### 반론들

**허구주의 (Field)**
```
과학은 수학 없이도 가능하다
수학은 유용한 허구일 뿐
```

**도구주의**
```
수학은 계산 도구
존재론적 개입 없이 사용 가능
```

**구조주의**
```
수학적 대상이 아닌 구조가 중요
수 자체가 아니라 수 체계의 구조
```

## 7. 다른 입장들 요약

### 직관주의 (Intuitionism)
```
- 수학은 정신적 구성
- 배중률 제한: ¬¬P ⊬ P
- 존재 증명은 구성적이어야
```

### 구조주의 (Structuralism)
```
- 수학은 구조에 대한 학문
- 개별 대상이 아닌 패턴/관계
- "2"는 자연수 구조에서의 위치
```

### 허구주의 (Fictionalism)
```
- 수학 문장은 문자 그대로 거짓
- 그러나 유용한 허구
- 소설과 유사
```

## 핵심 정리

1. **수 vs 숫자**: 추상적 대상 vs 표기 기호
2. **플라톤주의**: 수학적 대상 실재, 발견의 대상
3. **형식주의**: 기호 게임, 괴델에 의해 한계 노출
4. **프레게 논리주의**: 수학 = 논리, 러셀 역설로 좌절
5. **반심리주의**: 객관성, 규범성, 필연성, 무한성 논증
6. **필요불가결성 논증**: 과학에 필수 → 수학적 대상 실재
7. **대안들**: 직관주의, 구조주의, 허구주의
